# نظام إدارة عمليات المصنع (FOMS)

## وصف المشروع

نظام متكامل لإدارة عمليات التصنيع والتحكم في المصنع، يتبع نماذج CRUD مع واجهة مستخدم تفاعلية.

## التقنيات المستخدمة

- **Frontend**: React + Vite
- **Backend**: Express.js
- **Database**: SQLite
- **UI**: Custom design مع Glass morphism

## الوحدات الرئيسية

- **Production**: إدارة الإنتاج اليومي وتتبع العمليات
- **Machines**: إدارة الآلات ومتطلبات التوافق مع القوالب
- **Molds**: إدارة القوالب ومتطلبات الآلات (tonnage)
- **Materials**: إدارة المواد الخام
- **Users**: إدارة المستخدمين والأدوار
- **Maintenance**: صيانة الآلات
- **ParameterCheck**: فحص المعاملات
- **Reports**: التقارير والإحصائيات
- **Settings**: إعدادات النظام

## طريقة التشغيل

```bash
npm install
npm run dev
```

## التوافق التلقائي

النظام يدعم التوافق التلقائي بين القوالب والآلات بناءً على متطلبات الـ tonnage.
